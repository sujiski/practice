# 車両追従シミュレーション（C++）

## 概要

本プロジェクトは、C++で実装した1次元の車両追従シミュレーションです。
自車（Ego）が先行車（Lead）に対して、一定の車間距離を維持するようにPD制御を行います。

本プロジェクトの目的：

- C++によるコーディング力の向上
- ドメイン駆動設計（DDD）およびクリーンアーキテクチャの実践
- Simulink等のモデルベース開発で行っている制御ロジックをコードとして再現する

---

## 主な機能

- 1次元車両モデル（位置・速度・加速度）
- 環境モデル（自車 + 先行車）
- PD制御による車間距離制御
- 加速度の上下限制限（物理制約の考慮）
- 固定時間ステップによるシミュレーション

---

## アーキテクチャ

本プロジェクトは以下の設計方針に基づいています：

- ドメイン駆動設計（DDD）
- クリーンアーキテクチャ

### 設計上のポイント

- domain層は外部依存を持たない
- Controllerは状態を持たない（stateless）
- Environmentが状態の単一の管理主体（Single Source of Truth）
- const correctness を意識した設計

---

## 制御ロジック

加速度はPD制御で算出しています：

- `distance`：自車と先行車の距離
- `relative_velocity = v_lead - v_ego`

### 制御の意図

- P項：目標車間距離との差に応じた加減速
- D項：接近速度に応じた減速（衝突回避の安定化）
